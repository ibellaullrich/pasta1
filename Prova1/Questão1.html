1) (1.0)Explique detalhadamente o que ocorre durante o acesso a uma página na web, desde o momento em que o usuário digita a URL no navegador até a completa exibição do site. Descreva todas as etapas envolvidas nesse processo, incluindo a resolução de nomes de domínio, a troca de mensagens entre cliente e servidor, e os principais protocolos utilizados. Certifique-se de mencionar tecnologias como DNS, HTTP/HTTPS, TCP/IP, entre outras relevantes no processo. RESPOSTA: primeiro o usuario digita a url no navegador a partir disso o DNS transforma o nome do site em um indereço IP o navegador cria uma conexão TCP com o servidor, se for HTTPS acontece algo chamado handshake tls para garantir a segurança o navegador envia uma requisição HTTP, então o servidor processa e responde com HTML, CSS, JS. etc. o navegador recebe e interpreta o html, aplica o CSS e carrega recursos extras, por fim ele renderiza a pagina completa.
